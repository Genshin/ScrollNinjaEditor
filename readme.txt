◎マップエディター概要
　○操作方法
　　・クリック(オブジェクトボタン)	…　クリックしたオブジェクトを作成
　　・ドラッグ（オブジェクト）		…　オブジェクトの位置を移動
　　・右クリック（オブジェクト）	…　オブジェクトを削除

　　・クリック(レイヤ―ボタン)		…　クリックしたレイヤ―へ移動
　　・クリック(インポートボタン)	…　ファイルの読み込み
　　・クリック(アウトプットボタン)	…　ファイルの書き出し
　　・クリック(オブジェクト)　＋　Wキー …　クリック中のオブジェクトを前面に移動
　　・クリック(オブジェクト)　＋　Qキー	…　クリック中のオブジェクトを背面に移動
　　・スクロール			…　画像の拡大・縮小表示
　　・スペースキー　＋　ドラッグ	…　カメラ移動
　　・Zキー			…　前のレイヤ―へ移動
　　・Xキー			…　次のレイヤ―へ移動
　　・Cキー			…　作業レイヤ―の切替(Front or Back)
　　・Vキー			…　新規レイヤーの追加
　　・Bキー			…　選択レイヤーの削除
　　・Sキー			…　選択レイヤーを前面へ移動
　　・Aキー			…　選択レイヤーを背面へ移動
　　・Dキー			…　選択レイヤーのみ描画
　　・Fキー			…　全てのレイヤーの描画
